JavaScript Basic

JavaScript
原生JavaScript, 也有很多标准了, 主流的是ES5, ES6; 传统的js代码是跑在用户的浏览器,作为一种增强交互方式的脚本语言存在的,因此安全性很重要(不能主动访问用户电脑硬盘等);

质变 => NodeJS(后台) 阅读材料
近几年js的告诉发展起源于nodejs, nodejs使得js可以编写server端代码. nodejs虽然是单线程,但是js天生的非阻塞I/O,事件驱动模型,使得其仍然有很高的性能.
